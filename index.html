HTML

  • HTML creates the structure of webpages.
  • The head element contains information for the browser about the webpage.
  • The title elment gives the title of a webpage.
  • The style elment tells the browser what CSS to apply to the web page.
  • The base element tells which main URL belongs to a webpage.
  • The meta element gives various data about the webpage, like the viewport and character encoding.
  • The body element contains the information that the user sees.

CSS

  • The 'margin' property tells the amount of space that will be created surrounding the outside of an element.
  • The 'padding' property tells the amount of space that will be creating inside of the box of an element.
  • The 'border' property creates a colored box around a certain element.
  • The 'box-shadow' property creates the appearance of a shadow next to a certain element.

Git

  • git status - Checks what branch the user is currently in.
  • git checkout -b [branch-name] - Creates new branch and switches to said branch
  • git add [-A] - stages files the for next commit. -A flag means to add all the changes.
  • git commit [-m] Commits staged changes to the current branch. -m flag adds a message to the commit.
  • git pull origin [branch name] - Pulls code from the origin on github to the local machine.
  • git push origin [branch name] - pushes changes from the local machine to the origin on Github.
